To regenerate the help file and site:
1) Install Sandcastle: http://www.microsoft.com/downloads/details.aspx?FamilyId=E82EA71D-DA89-42EE-A715-696E3A4873B2&displaylang=en
2) Install the Sandcastle Help File Builder: http://www.codeplex.com/SHFB/
3) Copy both from the install location inside the moq\build folder.
3) Run Build.cmd
